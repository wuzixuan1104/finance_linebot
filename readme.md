# Welcome to 理財小精靈
## 說明
+ 透過這個理財小精靈，查詢各家銀行貨幣的當前匯率，並透過試算找出對自己最有利的購買時機

## 功能介紹
1. 查看各家銀行匯率
2. 換算銀行貨幣匯率
3. 查看各家銀行匯率歷史紀錄 (未完成)
